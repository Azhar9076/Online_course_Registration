Online Course Registration System

##  Project Description
The **Online Course Registration System** is a web-based application that allows students to register for courses easily through an online platform. It simplifies the manual registration process and provides a user-friendly interface for managing course enrollments.

---

##  Features
-  Student registration and login system  
-  Course selection and enrollment  
-  Data storage using database  
-  Easy navigation and user-friendly interface  
- Organized course information  

---

## Technologies Used

### Frontend
- HTML – Structure of web pages  
- CSS – Styling and layout  

### Backend
- PHP – Server-side scripting  

### Database
- MySQL – Data storage and management  

---

## How to Run the Project

1. Install a local server like XAMPP / WAMP  
2. Place the project folder inside `htdocs`  
3. Start Apache and MySQL  
4. Import the database file (`database.sql`) into phpMyAdmin  
5. Open browser and run:
